⭐ Featured Project — Ruman Store POS

The most technically complex project in this repository.

Ruman Store POS is a production-ready, offline-first mobile POS application built for retail shops. It features:

✅ Full offline operation via SQLite with zero data loss
✅ Cloud backup & restore via Supabase with a custom sync algorithm
✅ Complete POS billing, inventory management, and customer ledger
✅ Business analytics (daily/monthly reports, net profit tracking)
✅ QR/Barcode scanner, CSV export, Bluetooth printer support
